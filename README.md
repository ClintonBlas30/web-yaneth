# Sistema Web de Ejemplo para Sustentación

Este es un sistema web pequeño desarrollado por **MARTINEZ SUAREZ, Yaneth Liz** como evidencia de culminación de la carrera de **Computación e Informática**.

## Estructura del Proyecto

- `index.html`: Página principal con estructura comentada y personalizada.
- `styles.css`: Estilos principales, con comentarios para facilitar el aprendizaje.
- `img/`: Carpeta para imágenes (agrega aquí tus imágenes o foto personal).

## ¿Cómo probar el sistema?

1. Abre la carpeta del proyecto en VS Code.
2. Haz clic derecho sobre `index.html` y selecciona "Open with Live Server" (requiere la extensión Live Server).
3. El sitio se abrirá en tu navegador y se actualizará automáticamente al guardar cambios.

## Características

- Estructura HTML clara y comentada.
- Estilos modernos y responsivos.
- Formulario de contacto con mensaje de éxito visual (no envía datos reales).
- Sección personalizada "Sobre mí" con nombre, carrera y espacio para foto.

## Personalización

- Puedes reemplazar la imagen de ejemplo en la sección "Sobre mí" colocando una imagen en la carpeta `img` y cambiando el nombre en el HTML.
- Modifica los textos y colores en `index.html` y `styles.css` según tus necesidades.
